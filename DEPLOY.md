# Deploy to Vercel

## Quick Deploy

1. **Install Vercel CLI** (optional): `npm i -g vercel`
2. **Deploy**: Run `vercel` in this directory and follow the prompts
3. **Production**: Run `vercel --prod` for production deployment

## Deploy via Vercel Dashboard

1. Go to [vercel.com/new](https://vercel.com/new)
2. Import your Git repository (GitHub, GitLab, or Bitbucket)
3. Or drag-and-drop this folder for a one-time deploy
4. Vercel will auto-detect the static siteâ€”no build step needed

After deployment, you'll get a URL like: `https://your-project.vercel.app`
